# Circe's Garden v2

A narrative farming game about jealousy, guilt, and redemption, set in Greek mythology.

Platform: Retroid Pocket Classic (Android 14, 1080x1240, d-pad only)
Engine: Godot 4.5.1
Target Playtime: 65-90 minutes
