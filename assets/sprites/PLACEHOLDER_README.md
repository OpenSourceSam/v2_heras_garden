## Color Code

**Characters:**
- Circe (Player): Yellow/Gold `#FFD700`
- Scylla (Before): Light Purple `#DDA0DD`
- Scylla (After): Dark Purple `#8B008B`
- Hermes: Sky Blue `#87CEEB`

**Crops:**
- Moly: White `#FFFFFF`
- Nightshade: Purple `#9370DB`
- Lotus: Pink `#FFB6C1`
- Saffron: Red-Orange `#FF6347`

**Terrain:**
- Grass: Green `#4CAF50`
- Dirt: Brown `#8B4513`
- Tilled Soil: Dark Brown `#654321`
- Water: Blue `#1E90FF`

**UI:**
- Primary: Purple `#6A0DAD`
- Secondary: Gold `#FFD700`
- Background: Dark Purple `#2E003E`

## Usage

For now, use ColorRect nodes or simple Sprite2D with modulate colors.
Godot will render these as solid colors until actual sprites are imported.
